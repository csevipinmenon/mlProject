Student Exam Performance Indicator

About the Project

The Student Exam Performance Indicator is a machine learning–based web application that predicts a student’s Math score based on various factors such as gender, ethnicity, parental education level, lunch type, test preparation course, and reading & writing scores.

This project helps educators and analysts understand how socio-academic factors influence student performance. It shows how data science and machine learning can create predictive insights in education.

The model is trained in Python using scikit-learn and integrated with Flask to provide an interactive and real-time prediction experience.
